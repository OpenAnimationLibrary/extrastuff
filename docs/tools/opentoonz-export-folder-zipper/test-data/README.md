# Test data notes

Use an OpenToonz folder produced by:

```text
File > Export > Export Scene
```

Then select the exported folder in the web app.

A minimal exported scene folder should produce a summary similar to:

```text
Likely type: OpenToonz export
Scene files: 1
Project XML files: 1
Drawing/level files: 1+
Palette files: 1+
```

The generated ZIP should preserve the top-level exported folder and its subfolders.
