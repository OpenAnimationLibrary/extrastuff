Pan

A camera movement in which the camera scans an area by moving horizontally.

Path

A point or a series of points (spline) on which all objects must be assigned. A path may have more than one object.

Patriarch

The first bone added in a model.

Pivot

The point about which a group of control points move.

Polygonal Output

The ability to export a patch-based model or animation to several polygonal formats.
