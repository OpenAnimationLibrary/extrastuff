Attaching Control Points

To attach one control point to another control point, drag one control point over the control point you want to attach to, then, while keeping the mouse button pressed, press the right mouse button (Macintosh users press the Tilde (~) key).

When multiple control points are attached, only a single control point is visible, even though many splines may run through it. Moving such a control point will move all of the splines as one.
