Muscle motions allow you to change the actual surface contour of a model over time by altering the control points at keyframes. Muscle motions are unique because they allow the surface of a model to bend and flex as if it were a pliable material. This is important when using techniques like squash and stretch, and when doing precise movements like those needed in facial animation.

When you build a model, every control point is assigned a unique identity number called the "CP#". Once a model is saved, every control point will always retain its CP#, even if other control points are added to or deleted from the model later. This means that muscle motions created for the segment will still be valid even if you change the model, of course, muscle movements assigned to a control point that has been deleted will be ignored, and new control points will have no muscle movement associated with them.

To access Muscle Mode, right click the Action list for the character you wish to create an action for and select New Action, or click on an existing action. Once the Action is highlighted, click the Muscle Mode button.
