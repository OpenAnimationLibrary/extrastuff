Dopesheet files are created outside MH3D, either with a text editor or with a utility program. They are simply ASCII text files that contain a list of Pose names and frame numbers. When a Dopesheet is applied to a model, the software uses the Pose name to find the appropriate Pose embedded in the model. If a Pose is not found, it will be ignored. It is, therefore, very important to choose Pose names carefully and consistently. You will likely want to use a standard set of phoneme mouth shape poses for doing lip sync.

Dopesheet files (*.DPS) have the following format. They start with a [DOPESHEETFILE] and end with a [ENDDOPESHEETFILE]. The middle is comprised of Dopeframe definitions which start with [DOPEFRAME], and end with [ENDDOPEFRAME]. The following example shows a Dopesheet with two dopeframes, one telling frame 1 to use Pose1, and the other telling frame 15 to use Pose2.

[DOPESHEETFILE]

[DOPEFRAME]

Name=pose1

Frame=1

[ENDDOPEFRAME]

[DOPEFRAME]

Name=pose2

Frame=15

[ENDDOPEFRAME]

[ENDDOPESHEETFILE]
