Clicking this button will paste whatever is in the "paste" buffer into the active window. Indicate where you want to place the control points being pasted with the mouse, and click the mouse button to perform the paste operation. The new points will remain grouped and can be dragged with the mouse by holding the button down.

The Paste command can also be accessed from the Edit menu.

The default keyboard equivalent for Paste is Ctrl+V.
