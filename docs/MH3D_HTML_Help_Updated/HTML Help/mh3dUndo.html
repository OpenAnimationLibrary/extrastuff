If you are in the process of using the modeler and realize you have made a mistake, you can undo by clicking this button. The number of undo levels can be set in the Tools » Options » Modeling tab dialog. Undo defaults to 10 levels of undos.

The Undo feature can also be accessed from the Edit menu.

The default keyboard equivalent for Undo is Ctrl+Z.
