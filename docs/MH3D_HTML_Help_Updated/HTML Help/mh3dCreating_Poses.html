There are two methods that can be used to create Poses. To use the first method, right-click a model in the Project Workspace and select New, Pose. A window similar to an Action window will open that will allow you to manipulate bones or groups of points (Muscle mode) and save the final positions as a Pose.

The second method is to select Create Pose from the Action drop down menu while in Action. The advantage of making Poses in Action is that you can make keframes as you go, rather than starting from the a default position each time.

Many times, a Pose contains Muscle groups that use the default position (no motion) for some points in the group. To properly save a Pose like this, you need to choose make keyframe from the Edit menu to create a keyframe for all points in the group, including those that do not move.
