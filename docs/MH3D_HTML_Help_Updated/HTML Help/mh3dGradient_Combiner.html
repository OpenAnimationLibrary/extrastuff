A Gradient combiner has two child nodes, between which it creates a gradation. This gradation can be a linear gradient from a given starting point to an ending point. Or it can have Edge Threshold, causing the color to gradate towards the edges of the object. There can also be a combination of both.

The start and end position determines the points from which the gradient will start with the first color, and end with the second color.

The Edge Threshold value causes the edges and the face of your object to gradate between the two colors.
