A color map is merely taking an image file and mapping it on your object. The colors in the image file are used instead of the color of the model. The Value determines the amount of color that comes from the image file versus the objects color. For example, a value of 75 would take 75% of the image map color and 25% of the underlying segment color. This number should range from 1 to 100.
