To achieve a glowing affect on a particular area of an object, use an Ambiance map. This technique works well for making the appearance of lit windows on a building or spacecraft. It is generally used in conjunction with a Color map, since the Ambiance map simply tells how bright pixels are. Ambiance maps are grayscale images where white is 100% ambient, and black is 0%.

The value determines how much effect the grayscale image has on the surface. A higher number would make the surface appear more ambient. Reasonable values for Ambiance maps are from 1 to 1000%.
