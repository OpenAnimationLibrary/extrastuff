Background Color

The color in a bitmap that is ignored when the bitmap is used. For example, picture a decal of a logo on a white background. By setting the background color to white, any white pixels in the image are ignored, leaving just the logo image.

Bias

The orientation of the curvature of splines.

Boolean Cut

When modeling, sometimes it is easier to remove space than it is to add it. Let’s use an apple with a bite out of it for example. To model the apple originally with the bite already out of it is much more difficult than modeling the apple” and “bite” as separate segments. After modeling both, you subtract the bite from the apple. This is called a “Boolean” cut. In a Boolean cut, the subtracted part leaves behind its own surface characteristics, so in the apple, the ‘bite’ segment would look like the inside of an apple, while the apple segment would still appear shiny red.

Bound

The fastest drawing mode, it draws only a bounding box to represent each Segment.

Bound Group

This function allows you to drag a box to designate which control points you wish to select.

Branch

Refers to a part of the hierarchical chain of a character. For example, a bicep, forearm, and hand would make a branch.

Bulb

The default light type. Bulbs are omnidirectional, meaning they cast light rays in all directions.

Bump Map

A grayscale image that uses the intensity of adjacent pixels to imitate surface irregularities. White areas of the map appear raised, while darker areas appear lowered.
