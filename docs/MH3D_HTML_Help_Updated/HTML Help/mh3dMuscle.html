Muscle

Muscle mode can be accessed once a new or existing action file has been opened or created. Muscle mode can also be used in a Choreography window by selecting an object, then clicking the Muscle mode button.

Muscle mode allows you to create actions that require the manipulation of the surface of an object. Here, you can translate, or rotate a control point or a group of control points over time. Lip synching is an example of muscle motion.
