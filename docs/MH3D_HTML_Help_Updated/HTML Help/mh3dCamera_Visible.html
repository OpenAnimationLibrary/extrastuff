The Camera Visible checkbox indicates which cameras in a scene are visible when multiple cameras are used. This checkbox will help to keep your work area clean when working on complex chorographies. To return a given camera to a visible state, select the camera in the Project Workspace, and click in the Camera Visible checkbox on the Properties Panel.
