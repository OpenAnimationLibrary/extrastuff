The Cone Visible checkbox indicates which camera cones in a scene are visible. This checkbox will help to keep your work area clean when working on complex chorographies. To turn a given cameras cone back on, select the camera by clicking it in the Choreography window or the Project Workspace, and click in the Cone Visible checkbox on the Properties Panel.
