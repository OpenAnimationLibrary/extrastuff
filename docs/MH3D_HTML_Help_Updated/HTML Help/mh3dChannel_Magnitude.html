When a control point is clicked inside of a Channel editing window, the Magnitude can be altered by adjusting the Yellow handles on either side of the control point. The values will change on the Properties Panel to reflect the changes you are making. If necessary, you can manually enter Magnitude values in the Mag In and Mag Out fields. They control the Magnitude value of the selected spline as it goes in to the control point, and as it comes out of the control point, respectively.
