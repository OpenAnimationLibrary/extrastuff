Usually, an image map is fixed, meaning it stays attached to the same place on the model no matter how the model translates, scales, or rotates in the final animation. An image map that is not fixed is called an environment map because the model appears to be reflecting the environment around it as it moves. The image projected onto the model always stays oriented facing the observer as the object rotates.

The best way to think of an environment map is to imagine yourself standing in front of a movie screen.The picture would be projected onto your body, and as you moved, the part of the projected picture falling on you would change.

To apply an image as an environment map, create a New Material, expand its tree by clicking the plus (+) sign to the left of the item name, then right click the Attribute node and select Import Environment Map from the available menu. The material can now be applied to an object just as any other material.
