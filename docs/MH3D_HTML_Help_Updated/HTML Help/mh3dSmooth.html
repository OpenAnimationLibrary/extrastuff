Initially, control points on a spline are smooth, meaning that the spline remains continuous as it enters and leaves a control point.

The magnitude and bias of curved splines is adjusted once, and affects both sides of a spline, since a curved spline is continuous. Curved splines are excellent for constructing organic looking models.

All grouped control points are affected when the Smooth button is clicked. The state of a selected control point will show by the appropriate Peak or Smooth button being down.

The default keyboard shortcut key for Smooth is O.

To change a peaked control point into a smooth one, select the point by clicking on it with the mouse. The selected point and associated spline will turn green.

Click the Smooth button on the Tool Panel.
