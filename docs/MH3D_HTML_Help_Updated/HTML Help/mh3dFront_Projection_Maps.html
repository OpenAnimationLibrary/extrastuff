Front projection is a classic technique for mixing computer animation with live action. Rough geometry is constructed to represent items in the live action scene, then an image is projected into the scene and applies only to the specified models.

To specify a projection map in Directing Mode, right click the camera, and load the live action frames by selecting Import Rotoscope from the available menu. The rotoscope will appear in the Project Workspace, and the Properties Panel will update to reflect the available options. To specify the map as a front projection type, click in the Front Projection Map checkbox on the Properties Panel.

The Shortcut to Model item for each model to which the map should be applied can then be selected in the choreography section of the Project Workspace. To specify the model as a front projection target, click in the Front Project Target checkbox on the Properties Panel.
