There are several Grouping Commands that make it easier to select those control points that you want. They can all be selected from the Edit menu.

Group Connected :

All points on the same graph as the currently selected point will be added to the group regardless of their previous grouped or ungrouped state.

Complement All :

All editable points on all graphs will be added to the group or removed from the group. The points will possess the opposite state as they had before the command.

Deselect All :

All points on all graphs will be de-selected.

Select All :

All editable points on all graphs will be grouped.
