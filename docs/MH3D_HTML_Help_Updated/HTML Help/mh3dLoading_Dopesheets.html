Think of a dopesheet as something you apply to a model from a choreography. It’s very similar to applying a regular action. Once you have an open choreography with a model inserted in it, you are ready to apply a dopesheet. Move the mouse to the Project Workspace and expand the choreography so the model is listed. Now, right click (Command-click on the Mac) the model so the popup menu appears. Now, select “Apply Dopesheet”. A file dialog will appear. Select the dopesheet file you want to apply, and it will load and be applied to the model.
