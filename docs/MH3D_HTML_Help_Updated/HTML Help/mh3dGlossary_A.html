Absolute Motion

The direction, movement, and physical location of a character in world coordinates

Actions

A combination of Skeletal and Muscle motions.

Alpha Channel

The 8 bits of a 32 bit image that contain transparency information. This information can be used to selectively remove certain parts of an image.

Ambiance

A value which specifies how illuminated a bones geometry will appear even with no direct light shining on it.

Ambiance Map

A grayscale map used to achieve a lighting effect on a particular area of an object, where white designates a 100% ambient value, and black designates 0% ambiance. This technique works well creating lighted windows on a building.

Anchoring

A constraint used when, for example, a characters waist is to be moved while the feet remain on the floor.

Attribute

Refers to colors, specularity, roughness, transparency, etc.

Auto-Tracking

A constraint that keeps a character correctly oriented on a path. Using a car as an example, auto-tracking would direct the car to always face in the direction it is traveling. Without Auto-tracking, the car would face in the direction in which you placed it.
