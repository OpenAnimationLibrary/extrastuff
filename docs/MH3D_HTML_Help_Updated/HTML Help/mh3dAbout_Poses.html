Poses are predefined bone and muscle positions that can be stored with a character or object. A pose can be made from a bone action, a muscle action, or both. They are created soon after the character is modeled and stored within the model file.

Essentially, Poses are one frame actions for positions that you will use frequently. The most common use for poses is for the basic mouth shapes used for lip synching. You will use poses during the process of animating, so it is good to think of poses as part of the model. When working with a model that has poses, a list of the embedded poses can be opened in the Project Workspace. Any pose can then be dragged and dropped into an action at any frame.
