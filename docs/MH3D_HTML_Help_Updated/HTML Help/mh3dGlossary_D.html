Decal

The process of taking a 2-D image and gluing it onto a 3-D object. There is no limit to the number of decals that can be applied to a model.

Diffuse Map

A grey scale map used to cause blemishes on the surface of an object. As colors get darker on the Diffuse map, the underlying color of the object gets darker.

Displacement Maps

Unlike Bump maps, which only give an object the illusion of bumpiness, Displacement maps actually affect the object’s surface. Similar to Bump maps, gray scale images are ideal for displacement maps. However, contrary to Bump maps, the grey value determines the amount of displacement, (Bump maps look for changes in value rather than the value itself): “0” being no displacement, while “255” equals 255 units of displacement (the same units used by the rest of the program). The “Percent” of displacement will raise or lower the displacement value by the specified percent (50% with half the value, while 200% will double it). Negative “Percent” values will cause the displacement to occur in the opposite direction. As a bone scales, so will its displacement.

Distance to Fall-Off

This function controls the distance from the light at which fall-off begins to occur. Objects closer to the light than this distance receive the full intensity of light. As objects get farther than this distance, less light falls on them.

Dynamics

Also known as rigid body dynamics. Places the burden of computing the position and orientation of any figure on the software.
