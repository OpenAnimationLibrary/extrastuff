Click this button to save current changes to your work. As you are working, the software keeps track of which files have not been saved, and marks them with an asterisk (*) after their name. When you save, all of the asterisks will clear from the file names.

The default keyboard equivalent for Save is Ctrl+S.

Remember to save often.
