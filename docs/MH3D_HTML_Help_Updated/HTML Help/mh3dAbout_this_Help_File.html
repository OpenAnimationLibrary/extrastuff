About this Help File

How to use this recovered and updated edition.

This collection originated as the HTML help for Martin Hash’s 3-Dimensional Animation, an early predecessor of Animation:Master. It has been preserved because many explanations of splines, patches, control points, bones, actions, constraints, and choreography remain historically and technically useful.

What was updated: Missing topic filenames were restored, navigation was repaired, the pages were converted to modern UTF-8 HTML, and a recovery page was added for unavailable destinations. The original illustrations and most instructional text remain intact.

Mouse and platform terminology

The original text refers to “PC,” “Macintosh,” multiple mouse buttons, and Command-clicking. In this edition, interpret right-click as the operating system’s configured secondary click. On a trackpad, that may be a two-finger click, a corner click, Control-click, or another configured gesture.

Historical accuracy

Screen images and procedures describe the older program. Current Animation:Master releases can differ in appearance, shortcuts, menu placement, supported formats, and available features. No automatic substitution has been made where doing so could change the technical meaning of an older procedure.

Navigation

Every page now has links to the Contents, the alphabetical Index, this page, and current resources.
