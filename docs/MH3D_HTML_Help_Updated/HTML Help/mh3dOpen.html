Clicking the Open button will allow you to open an existing project, choreography, model, action, or material.

Existing Projects have a .prj extension on the filename.

Existing Choreography files have a .cho extension on the filename.

Existing Model files have a .mdl extension on the filename.

Existing Action files have a .act extension on the filename.

Existing Material files have a .mat extension on the filename.

The default keyboard equivalent for Open is Ctrl+O.
