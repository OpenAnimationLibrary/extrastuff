Scale

The shrinking or enlarging of objects.

Screen Axis

The 2-Dimensional axis represented by the computer screen.

Script

A series of Actions associated with a Figure.

Sculpture

A window in which you create an object.

Segment

The most elementary three dimensional object.

Segment Drawing Modes

The different modes that each segment can be drawn in (bound, vector, curved).

Skeletal Motion

The translation, scaling, and rotation of Segments within a Figure, which can be stored for future use by other Figures.

Skin

The ability to join segments together, making your Figure look as if it were made out of one segment when it is actually many.

Snapshot

Function which allows you to save a customized screen layout.

Specular Map

A grey scale image used to create highlights on an object. Darker colors on the map cause the specularity of the underlying object to lessen. A Specularity map will override the base specularity of the object.

Spherical

A mapping technique applicable to spherical shapes, such as a ball.

Spherical Combiner

A material combiner which alternates material nodes back and forth.

Spline

The basis for patches, Splines are made from control points.

Spotlight

Also known as a “spot”. A unidirectional light source.

Stereo Options

Option settings to render an image or sequence of images from slightly different views to be used in creating 3D images.

Stereo Spacing

Allows you to adjust the distance between the two “eyes” or views in a 3D image.

Stride Length

Function to determine how far a Figure must move in each frame to keep its feet affixed to the ground and prevent it from ‘slipping’ as it walks.

Sun

A unidirectional light source whose intensity remains constant.
