Once the properties values are set to your liking and you have positioned the decal on your object the way you want, simply right click on the map and choose Apply from the popup menu. Beneath the name of the decal in the Project Workspace, the word “Stamp” and an embedded number will appear. You can rename each application of the decal, making it easier to keep track of how your image maps are applied. Clicking on any one of the Stamp tags and pressing the Delete key will remove that occurrence of the map from the surface of the model. To remove a decal completely, simply select it’s shortcut in the Project Workspace and press the Delete key.

Images with embedded frame numbers at the end of their names (i.e. Wave0001, Wave0002) used as a decals will be considered to be part of a sequence. Sequence controls for animated image maps are on the Properties Panel Sequence tab.

Right clicking the image map in the Modeler window and selecting Edit from the popup menu will launch the paint program that is associated with the image file extension through Windows.
