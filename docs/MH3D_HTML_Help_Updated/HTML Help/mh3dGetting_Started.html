Getting Started

A practical path through the recovered documentation.

Creating and animating characters takes practice. Begin with a small exercise rather than a large production, and learn the vocabulary and file structure before heavily customizing the interface.

  1. Read About Navigating Around a Window and practice moving, turning, and zooming the view.
  2. Work through Modeling Simple Objects to learn control points, splines, patches, and the primary modeling tools.
  3. Continue with the quick-start sections for materials, bones, actions, and directing.
  4. Save frequent numbered versions of your project and keep external images, sounds, and models together in an organized project folder.
  5. When a command or screenshot differs from the current release, use the current resources page to reach the modern manuals and user forum.
Modern release note: This recovered manual predates Animation:Master 19.5g. Treat it as a guide to enduring concepts rather than an exact map of every current menu, property, or toolbar.

Most importantly, experiment. Short tests reveal how splines deform, how constraints interact, and how lighting and render settings affect the final image much faster than a large scene does.
