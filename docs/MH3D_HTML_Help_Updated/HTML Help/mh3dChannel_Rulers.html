Rulers are drawn with frames on the horizontal axis and values on the vertical axis. The spacing of each is chosen as small as possible while preventing overcrowding of adjacent numbers. Possible spacing for the frames are 1,2,5,10,15,20,30,50,60 and 100. The possible spacing for the values are .001, .005, .01, .05, .1, .5, 1, 2, 5, 10, 25, 50, 100, 200, 500, 1000, 2500, 5000, 10000.
