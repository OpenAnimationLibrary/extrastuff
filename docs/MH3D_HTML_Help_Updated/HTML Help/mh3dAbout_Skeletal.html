Skeletal motions allow you to change the rotational movement that occurs at a figure’s joints such as bending an elbow or knee. Skeletal motions are reusable from figure to figure and include rotation, translation, and scaling of the control points associated with each bone.

To access Skeletal Mode, right click the Action list for the character you wish to create an action for and select New Action, or click on an existing action. Once the Action is highlighted, click the Skeletal Mode button.
