Bones mode can be accessed once you have chosen to create a new model, or opened an existing one for modification.

Bones mode is used to easily add or modify a hierarchical “skeleton” in a character. This is done by assigning new bones, attaching or detaching existing bones, naming bones, and setting bone attributes.

The skeletal hierarchy can easily be modified by dragging and dropping the bones in the Project Workspace.
