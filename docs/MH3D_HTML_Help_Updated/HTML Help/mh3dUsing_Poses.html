Poses give you an easy shortcut to animating common motions, particularly facial expressions. Once you have poses created for a model, you can easily insert them into an action using drag and drop. A pose will only affect the grouped points and the bones that were selected when the pose was saved.

Create or load an existing action, and open an Action window. Expand the list of the model’s poses in the Project Workspace, and move to the appropriate frame. Click on the pose you wish to insert, and with the mouse button still down, drag the pose into the Action window and release the mouse button. The muscle and bone position information will be inserted into the action. If just the mouth was grouped when the pose was saved, then only the mouth will be affected. If the Action Window is in Muscle mode when you drop a Pose, it will only affect the selected group. If the Action Window is in Skeletal mode when you drop a Pose, the Pose will be applied to your model depending on the key mode buttons selected on the Frame toolbar. If you want the complete Pose to be applied, make sure that Key Skeletal Translation, Key Skeletal Scale, Key Skeletal Rotation and Key Model are selected.

Poses can also be used in a choreography window. This is useful when you are animating a character while you are in Directing Mode and do not wish to save a separate action file. Rather than dragging the Pose into the window as in Action, simply drag the Pose onto the models shortcut in the appropriate choreography folder.

Inserting Poses into an action is just an intermediate step. You will usually change muscles and bones after you have inserted Poses. You may also want to tweak muscle points after Poses have been inserted. Perhaps you are using the oo” pose, but you want to exaggerate it more. After dragging it to the appropriate frame, you can group some of the points and pull them out more. You can also animate the cheeks and areas around the mouth to give the face more expression.
