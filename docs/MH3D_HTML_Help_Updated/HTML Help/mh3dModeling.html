Modeling

When a new model is started, an empty window opens and you are automatically placed in modeling mode. From this point, you can use the modeling tools to create a new character or modify an existing one by adding, translating, rotating, subtracting, or changing the bias of a control point or group of control points.

If you are working on an existing model elsewhere in the program, you can switch back to modeling mode by clicking this button. The current window will change appropriately.
