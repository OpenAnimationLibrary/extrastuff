Skeletal

Skeletal mode can be accessed once a new or existing action file has been opened or created. Skeletal mode can also be used in Directing mode by selecting an object, then clicking the Skeletal mode icon.

Skeletal mode allows you to create actions that require the manipulation of the bones placed in an object. Individual bones can be locked, translated, scaled, and rotated over time. Skeletal mode is also where stride length is set to help keep characters feet from slipping along the path they are traveling on. Walking is an example of skeletal motion.
