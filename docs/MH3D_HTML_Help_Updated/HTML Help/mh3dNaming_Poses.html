If you expand the model in the Project Workspace, you will see an item called Poses. If you expand the poses list, you will see the complete list of poses for that model. Newly created Poses will have a generic name. You can name a pose by clicking once on the item name to select it, and a second time to enter data. Poses should be named immediately after creation.

The name of a pose is very important. Choosing short but descriptive names will help you later when it is time to drop poses into an action.

Pose names are also used by dopesheets. A dopesheet does not contain pose data, since it can work with any number of models. It refers to a model’s poses by name. This means that if you want to make mouth poses for a group of characters, you need to use the same pose names for all of the characters. You may make a mouth pose for the sound “oo” for several characters. They may differ somewhat based on the difference between the character models. But the name of the pose should be “oo” for all of the characters. This is why poses should be reserved for simple but common positions that will be mixed in with other motion for the final animation.
