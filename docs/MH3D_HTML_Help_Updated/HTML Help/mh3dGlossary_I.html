Image Maps

A graphic (usually .TGA or .PCX on a PC, or .TGA and .PICT for a Mac) used to create surface details on an object. Animated images can have the play modes adjusted on the Properties Panel.

Image Mapping A mapping technique that will map the image onto the

model and use the colors in the image file instead of the color of the bone.
