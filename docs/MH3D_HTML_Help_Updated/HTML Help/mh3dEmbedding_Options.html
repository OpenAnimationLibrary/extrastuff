On the File tab for each Model, Material, Pose, etc., there are embedding options that allow you to specify whether or not the files are embedded within the project you are working on, or whether they are saved separately. This is a significant option, since saving objects outside of a project make them more accessible, but harder to keep organized.

The default is to save the support files within a project. To change this, simply click the File tab after selecting an object, then click the Browse button. You will be able to specify where on your hard drive the file is placed.

Note:This just tells the software where to save the file. It won’t actually be saved until you click the Save button on the Standard Toolbar, or select Save from the File menu.
