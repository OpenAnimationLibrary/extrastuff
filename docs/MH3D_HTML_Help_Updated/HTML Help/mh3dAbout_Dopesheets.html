Another use for poses is dopesheets. A dopesheet is created outside of MH3D, and is essentially a special kind of action that is made up of pose names and frame numbers. A dopesheet is created by using a text editor or sound synching utility.

This allows you to create an action comprised of basic mouth shapes and expressions that is independent of a specific character. The only requirement is that you created the appropriate basic poses for all of the models that use that dopesheet. You may have a cast of characters that have very different mouth designs. Among three characters the pose for the “oo” shape may have a different scale and different number of points, but all are recognizable as the “oo” shape. Using a dopesheet, you can have your cast of characters all sing in a chorus together. In this way, poses can be used as a building block for more complex actions.

The most common use for dopesheets is for lip sync actions. Since you can layer actions on the same model, you can use a dopesheet for just the mouth motions, but use a regular action for animating the rest of the face.
