Introduction

An updated introduction to this recovered help edition.

About this archive: These pages were written for an early release sold as Martin Hash’s 3-Dimensional Animation. The software later became Animation:Master. Core ideas such as spline-and-patch modeling, bones, actions, choreography, decals, materials, cameras, lights, and rendering remain useful, but the modern interface contains additional tools and may use different menu names.

Animation:Master is an integrated 3D character-animation system. A typical project combines models, surface properties, materials and decals, skeletal rigs, reusable actions, choreography, cameras, lights, and rendered output.

The software’s distinctive modeling workflow uses intersecting splines and curved patches rather than relying only on polygonal meshes. This help set explains that workflow from basic navigation through modeling, texturing, rigging, animation, directing, and rendering.

Use the Contents to study topics in sequence or the alphabetical Index to locate a specific command or concept.
